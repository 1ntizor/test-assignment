# Команды поиска в Ubuntu

## Основные команды grep:
```bash
# Рекурсивный поиск
grep -r "текст" /путь/

# С номерами строк
grep -rn "пример" /папка/

# Только имена файлов
grep -rl "config" /etc/
```

## Примеры использования:
```bash
# Поиск в текущей директории
grep -r "password" .

# Поиск с игнорированием регистра
grep -ri "database" /var/www/
```
